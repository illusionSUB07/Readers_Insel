# Readers_Insel

Author: Daniyal Khan

**Items' Category:** (BOOKS)

Readers_Insel is a web application dedicated to bookworms. Users can add, edit, and remove books, making it a valuable platform for avid readers.

- **Back-End:** Node.js, Express.js.
- **Database:** MySQL.
